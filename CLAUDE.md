# CLAUDE.md — Mini Library Management System

## Төслийн тойм
Java Maven дээр суурилсан номын сангийн удирдлагын систем.
Book, Member, Loan гэсэн 3 үндсэн domain-тай.

## Build команд

### Compile хийх
```bash
cd partB/library-system
mvn compile
```

### Тест ажиллуулах
```bash
cd partB/library-system
mvn test
```

### Ажиллуулах
```bash
cd partB/library-system
mvn exec:java -Dexec.mainClass="library.Main"
```
