# Mini Library System

## 📌 Project Overview
Mini Library System is a simple web-based application for managing books, members, and loan tracking in a small library.

## 🎯 Purpose
- Manage book inventory
- Manage library members
- Track book borrowing and returning

## 🧩 Main Features
1. Add new books to the system
2. Register library members
3. Borrow books
4. Return books
5. Search books by title/author

## 📦 Scope
- Backend API based system
- Simple database (SQLite)
- No complex UI required (optional simple frontend)

## 🚫 Out of Scope
- Payment system
- Advanced authentication (OAuth, etc.)
- Cloud deployment
