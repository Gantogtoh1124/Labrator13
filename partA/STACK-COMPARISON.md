# Stack Comparison

## 1. Node.js + Express + SQLite
- Pros:
  - Easy to learn
  - Fast development
  - Lightweight
- Cons:
  - Not best for large enterprise systems

## 2. Java Spring Boot + MySQL
- Pros:
  - Very stable
  - Enterprise level
- Cons:
  - Complex setup
  - Slower development

## 3. Python FastAPI + SQLite/PostgreSQL
- Pros:
  - Very fast API development
  - Clean syntax
- Cons:
  - Smaller ecosystem than Node

## 🎯 Final Choice
We choose: **Node.js + Express + SQLite**

## 📌 Reason
- Simple and fast to implement
- Good for learning API structure
- Easy integration with testing and AI workflow
