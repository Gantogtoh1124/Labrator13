# ADR-001: Technology Stack Selection

## 📌 Context
We need to choose a technology stack for building a Mini Library System.

## 🔍 Options Considered
1. Node.js + Express + SQLite
2. Java Spring Boot + MySQL
3. Python FastAPI + SQLite

## ⚖️ Decision
We choose **Node.js + Express + SQLite**

## 📌 Reason
- Simple and easy to implement
- Fast development speed
- Good for small-to-medium applications
- Suitable for learning API design

## 📉 Consequences
- Not suitable for very large systems
- Requires manual scaling if system grows
